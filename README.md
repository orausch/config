# other things

## i3-lock automatic
Copy `i3lock.service` into `/etc/systemd/system/`
run `systemctl enable i3lock.service`

## dmenu wifi
install https://github.com/firecat53/networkmanager-dmenu to /usr/bin/
install dmenu .desktop file
